Project 5: Text Processing with SpaCY

About the Project

This project focuses on text processing and word frequency analysis using the SpaCY model. The main goal is to fetch text from the given URL, print the statistics of the text, get word counts, and to print the top 10 most frequent words. Furthermore, we will remove stop words and punctuation to return a list of lemmatized words and their counts.

Sample Code

Fetching the text. For this project I used the first chapter of Stephen Chbosky's novel "The Perks of Being a Wallflower" for analysis. The code uses the requests library to fetch the text from the provided URL and checks for a successful response before returning the text content.

Word Counts Function

This function processes the fetched text to count word frequencies.

Top 10 Verbs Function

This function fetches and prints the top 10 verbs in the text.

Results

After running the code, we can see the statistics of the text, the word counts, the top 10 most frequent words, and lemmatization, which included the removal of stop words and punctuation. After removing stop words and punctuation, we can see a list of lemmatized words and their counts, which provides insight into the most frequently used words in the text without common filler words. The results show that the 15 most common words in the text are Michael, think, know, like, say, sister, try, want, go, school, tell, and dad. The words yield conclusions about the chapter and the books main themes which can be useful in the real world.
